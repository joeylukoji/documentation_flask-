1. Vue d'ensemble

Objectif

Après la connexion, l'utilisateur arrive sur une page tableau de bord qui :

  • Affiche son nom (depuis la session)
  • Est inaccessible sans être connecté
  • Propose un lien de déconnexion

Lien avec la page de connexion

/login (connexion OK)
    ↓ session créée
/dashboard (tableau de bord protégé)
    ↓ clic déconnexion
/login
Page Fichier Route
Connexion templates/login.html /login
Tableau de bord templates/dashboard.html /dashboard

Étape 1 — Prérequis

Avoir terminé le guide connexion :

  • app.py avec routes /login et /logout
  • session["user_email"] et session["user_name"] remplis à la connexion
  • templates/login.html

Ajouter le fichier dashboard à la structure :

Étape 2 — Page HTML du tableau de bord

Fichier templates/dashboard.html — HTML simple, sans CSS :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Tableau de bord</title>
</head>
<body>
    <h1>Tableau de bord</h1>

    {% with messages = get_flashed_messages() %}
        {% if messages %}
            <ul>
                {% for msg in messages %}
                    <li>{{ msg }}</li>
                {% endfor %}
            </ul>
        {% endif %}
    {% endwith %}

    <p>Bienvenue, <strong>{{ name }}</strong> !</p>
    <p>Vous êtes connecté en tant que : {{ email }}</p>

    <hr>

    <h2>Menu</h2>
    <ul>
        <li>Mon profil</li>
        <li>Mes tâches</li>
        <li>Mon emploi du temps</li>
    </ul>

    <p><a href="{{ url_for('logout') }}">Déconnexion</a></p>
</body>
</html>

Variables utilisées

Variable Source Exemple
{{ name }} session["user_name"] Administrateur
{{ email }} session["user_email"] admin@example.com

Étape 3 — Protéger la page (login_required)

Sans protection, n'importe qui pourrait ouvrir /dashboard directement. On ajoute un décorateur :

from functools import wraps
from flask import session, redirect, url_for, flash

def login_required(f):
    @wraps(f)
    def wrapper(*args, **kwargs):
        if "user_email" not in session:
            flash("Veuillez vous connecter.")
            return redirect(url_for("login"))
        return f(*args, **kwargs)
    return wrapper

Comment ça marche

Requête GET /dashboard
        ↓
login_required vérifie session["user_email"]
        ↓
  ┌─────┴─────┐
  │           │
 absent     présent
  │           │
  ↓           ↓
/login    dashboard()

Étape 4 — Route Flask du dashboard

Ajouter dans app.py :

@app.route("/dashboard")
@login_required
def dashboard():
    return render_template(
        "dashboard.html",
        name=session.get("user_name"),
        email=session.get("user_email"),
    )

Explication

Élément Rôle
@login_required Bloque l'accès si non connecté
session.get("user_name") Récupère le nom enregistré à la connexion
render_template(...) Affiche dashboard.html avec les variables

Étape 5 — Lier login et dashboard

Voici le flux complet dans app.py (connexion + dashboard) :

from functools import wraps
from flask import Flask, render_template, request, redirect, url_for, session, flash

app = Flask(__name__)
app.secret_key = "changez-cette-cle-en-production"

USERS = {
    "admin@example.com": {"password": "admin123", "name": "Administrateur"},
}

def login_required(f):
    @wraps(f)
    def wrapper(*args, **kwargs):
        if "user_email" not in session:
            flash("Veuillez vous connecter.")
            return redirect(url_for("login"))
        return f(*args, **kwargs)
    return wrapper

@app.route("/")
def index():
    if "user_email" in session:
        return redirect(url_for("dashboard"))
    return redirect(url_for("login"))

@app.route("/login", methods=["GET", "POST"])
def login():
    if request.method == "POST":
        email = request.form.get("email", "").strip()
        password = request.form.get("password", "")
        user = USERS.get(email)
        if user and user["password"] == password:
            session["user_email"] = email
            session["user_name"] = user["name"]
            flash("Connexion réussie.")
            return redirect(url_for("dashboard"))
        flash("Email ou mot de passe incorrect.")
    return render_template("login.html")

@app.route("/dashboard")
@login_required
def dashboard():
    return render_template(
        "dashboard.html",
        name=session.get("user_name"),
        email=session.get("user_email"),
    )

@app.route("/logout")
def logout():
    session.clear()
    flash("Vous êtes déconnecté.")
    return redirect(url_for("login"))

if __name__ == "__main__":
    app.run(debug=True)
