Page de connexion avec Flask

Guide simple pour créer une page de connexion avec Flask (HTML sans CSS).

Suite : une fois la connexion en place, consultez documentation-dashboard.md pour le tableau de bord.

1. Vue d'ensemble

Objectif

Créer la page de connexion avec :

  • Formulaire email + mot de passe
  • Vérification des identifiants
  • Session utilisateur
  • Déconnexion

Fonctionnement

1. L'utilisateur ouvre /login
2. Il remplit le formulaire (POST)
3. Flask vérifie email + mot de passe
4. Si OK → session créée → redirection vers /dashboard
5. Si KO → message d'erreur

Étape 3 — Page HTML de connexion

Fichier templates/login.html — HTML simple, sans CSS :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Connexion</title>
</head>
<body>
    <h1>Connexion</h1>

    {% with messages = get_flashed_messages() %}
        {% if messages %}
            <ul>
                {% for msg in messages %}
                    <li>{{ msg }}</li>
                {% endfor %}
            </ul>
        {% endif %}
    {% endwith %}

    <form method="POST" action="{{ url_for('login') }}">
        <p>
            <label for="email">Email :</label><br>
            <input type="email" id="email" name="email" required>
        </p>
        <p>
            <label for="password">Mot de passe :</label><br>
            <input type="password" id="password" name="password" required>
        </p>
        <button type="submit">Se connecter</button>
    </form>
</body>
</html>

Points importants

Élément Rôle
method="POST" Envoie les données de façon sécurisée
name="email" Clé lue par Flask avec request.form.get("email")
action="{{ url_for('login') }}" Envoie le formulaire à la route /login
get_flashed_messages() Affiche les messages d'erreur ou de succès

Étape 4 — Code Flask (login / logout)

Fichier app.py :

from flask import Flask, render_template, request, redirect, url_for, session, flash

app = Flask(__name__)
app.secret_key = "changez-cette-cle-en-production"

USERS = {
    "admin@example.com": {"password": "admin123", "name": "Administrateur"},
}

@app.route("/")
def index():
    if "user_email" in session:
        return redirect(url_for("dashboard"))
    return redirect(url_for("login"))

@app.route("/login", methods=["GET", "POST"])
def login():
    if request.method == "POST":
        email = request.form.get("email", "").strip()
        password = request.form.get("password", "")

        user = USERS.get(email)
        if user and user["password"] == password:
            session["user_email"] = email
            session["user_name"] = user["name"]
            flash("Connexion réussie.")
            return redirect(url_for("dashboard"))

        flash("Email ou mot de passe incorrect.")

    return render_template("login.html")

@app.route("/logout")
def logout():
    session.clear()
    flash("Vous êtes déconnecté.")
    return redirect(url_for("login"))

if __name__ == "__main__":
    app.run(debug=True)

La route /dashboard sera ajoutée dans le guide tableau de bord.

Explication du code

Concept Rôle
app.secret_key Obligatoire pour utiliser les sessions
session["user_email"] Mémorise l'utilisateur connecté
request.form.get("email") Récupère la valeur du champ email
flash("message") Message temporaire affiché au prochain chargement
redirect(url_for("dashboard")) Redirige après connexion réussie
