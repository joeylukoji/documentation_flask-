Étape 3 — Frontend manuel (HTML)

Pas besoin de maîtriser le CSS ou le JavaScript. Créez simplement des fichiers HTML basiques que Flask affichera.

Principe

  1. Chaque page = un fichier .html dans templates/
  2. HTML simple : titres, paragraphes, formulaires, tableaux
  3. Pas de CSS requis pour apprendre Flask

Exemple — `templates/dashboard.html`

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Tableau de bord</title>
</head>
<body>
    <h1>Bienvenue, {{ name }} !</h1>
    <p><a href="{{ url_for('list_users') }}">Gérer les utilisateurs</a></p>
    <p><a href="{{ url_for('logout') }}">Déconnexion</a></p>
</body>
</html>

Exemple — `templates/users/list.html`

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Utilisateurs</title>
</head>
<body>
    <h1>Liste des utilisateurs</h1>
    <p><a href="{{ url_for('create_user') }}">+ Nouvel utilisateur</a></p>

    <table border="1" cellpadding="8">
        <tr>
            <th>Nom</th>
            <th>Email</th>
            <th>Rôle</th>
            <th>Actions</th>
        </tr>
        {% for user in users %}
        <tr>
            <td>{{ user.name }}</td>
            <td>{{ user.email }}</td>
            <td>{{ user.role }}</td>
            <td>
                <a href="{{ url_for('edit_user', user_id=user.id) }}">Modifier</a>
                <form method="POST" action="{{ url_for('delete_user', user_id=user.id) }}" style="display:inline">
                    <button type="submit">Supprimer</button>
                </form>
            </td>
        </tr>
        {% endfor %}
    </table>
</body>
</html>

Syntaxe Jinja2 utile

Syntaxe Usage
{{ variable }} Afficher une valeur
{% for item in liste %} Boucle
{% if condition %} Condition
{{ url_for('nom_route') }} Générer une URL Flask

Pour le CSS ou le JavaScript : ajoutez-les plus tard si besoin. Ce n'est pas nécessaire pour comprendre Flask.

Étape 4 — Connecter Flask (backend)

Objectif

Créer le serveur qui sert les pages HTML.

Fichier `app.py` — version de base

from flask import Flask, render_template, redirect, url_for

app = Flask(__name__)
app.secret_key = "changez-cette-cle-en-production"

@app.route("/")
def index():
    return redirect(url_for("dashboard"))

@app.route("/dashboard")
def dashboard():
    return render_template("dashboard.html", name="Administrateur")

if __name__ == "__main__":
    app.run(debug=True)

Lancer le serveur

python app.py

Ouvrir : http://127.0.0.1:5000

Comment Flask fonctionne

1. Navigateur demande  GET /dashboard
2. Flask appelle       dashboard()
3. Flask charge         templates/dashboard.html
4. Flask renvoie        HTML au navigateur

Concepts Flask essentiels

Concept Description
Route @app.route("/url") — associe une URL à une fonction
Template Fichier HTML dans templates/
render_template Génère une page HTML avec des variables
redirect Redirige vers une autre URL

Étape 6 — Modèle et base de données

Objectif

Comprendre comment stocker les utilisateurs de façon permanente dans une base de données, au lieu d'une simple liste Python qui disparaît au redémarrage du serveur.

Liste Python vs Base de données

Liste Python (managed_users = []) Base de données (SQLite)
Persistance Perdue au redémarrage Conservée sur le disque
Structure Dictionnaire {"name": "Alice"} Table avec colonnes
Recherche Boucle for manuelle Requête SQL automatique
Usage Apprentissage / tests Projet réel

Qu'est-ce qu'un modèle ?

Un modèle est une classe Python qui représente une table dans la base de données. Chaque attribut de la classe = une colonne de la table. Chaque instance = une ligne.

Modèle Python (class User)          Table SQL (users)
─────────────────────────          ──────────────────
User.id           ───────────────►  id
User.name         ───────────────►  name
User.email        ───────────────►  email
User.role         ───────────────►  role
User.password_hash ──────────────►  password_hash

Installation

pip install flask-sqlalchemy

Mettre à jour requirements.txt :

flask==3.0.0
flask-sqlalchemy==3.1.1

Configuration dans `app.py`

from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config["SQLALCHEMY_DATABASE_URI"] = "sqlite:///app.db"
app.config["SQLALCHEMY_TRACK_MODIFICATIONS"] = False

db = SQLAlchemy(app)
Paramètre Signification
SQLALCHEMY_DATABASE_URI Chemin vers le fichier SQLite (app.db créé automatiquement)
db = SQLAlchemy(app) Objet qui gère la connexion à la base

Définir le modèle `User`

Créer un fichier models.py (ou placer dans app.py) :

from app import db

class User(db.Model):
    """Modèle = table 'user' en base de données."""

    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(100), nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)
    role = db.Column(db.String(20), default="student")
    password_hash = db.Column(db.String(200), nullable=False)

    def __repr__(self):
        return f"<User {self.email}>"

Explication des colonnes

Colonne Type Rôle
id Integer, primary_key=True Identifiant unique, auto-incrémenté
name String(100), nullable=False Nom obligatoire, max 100 caractères
email String(120), unique=True Email unique (pas de doublon)
role String(20), default="student" Rôle par défaut : élève
password_hash String(200) Mot de passe hashé (jamais en clair)

Créer les tables

Au premier lancement, exécuter une seule fois dans un shell Python ou au démarrage :

with app.app_context():
    db.create_all()

Cela crée le fichier app.db et la table user avec toutes les colonnes.

Accéder aux données — requêtes de base

# Lire TOUS les utilisateurs
users = User.query.all()

# Lire UN utilisateur par son id
user = db.session.get(User, user_id)

# Chercher par email
user = User.query.filter_by(email="alice@school.com").first()

# Compter
total = User.query.count()

Ajouter, modifier, supprimer (niveau base)

from werkzeug.security import generate_password_hash

# CREATE — Ajouter
nouveau = User(
    name="Alice",
    email="alice@school.com",
    role="student",
    password_hash=generate_password_hash("motdepasse123"),
)
db.session.add(nouveau)
db.session.commit()          # Enregistre réellement en base

# UPDATE — Modifier
user.name = "Alice Martin"
db.session.commit()

# DELETE — Supprimer
db.session.delete(user)
db.session.commit()

Important : db.session.add() prépare l'ajout, mais c'est db.session.commit() qui écrit dans la base. Sans commit(), rien n'est sauvegardé.

Routes d'authentification

from flask import session, redirect, url_for, request, flash
from functools import wraps

USERS = {
    "admin@example.com": {"password": "admin123", "name": "Administrateur"},
}

def login_required(f):
    @wraps(f)
    def wrapper(*args, **kwargs):
        if "user_email" not in session:
            flash("Connectez-vous pour continuer.")
            return redirect(url_for("login"))
        return f(*args, **kwargs)
    return wrapper

@app.route("/login", methods=["GET", "POST"])
def login():
    if request.method == "POST":
        email = request.form.get("email", "").strip()
        password = request.form.get("password", "")
        user = USERS.get(email)
        if user and user["password"] == password:
            session["user_email"] = email
            session["user_name"] = user["name"]
            return redirect(url_for("dashboard"))
        flash("Identifiants incorrects.")
    return render_template("login.html")

@app.route("/logout")
def logout():
    session.clear()
    return redirect(url_for("login"))

@app.route("/dashboard")
@login_required
def dashboard():
    return render_template("dashboard.html", name=session.get("user_name"))

Points clés

Élément Rôle
session Mémorise l'utilisateur connecté
@login_required Bloque l'accès si non connecté
request.form.get() Récupère les données du formulaire POST
flash() Affiche un message temporaire

Étape 8 — CRUD utilisateurs

Objectif

Gérer les utilisateurs avec le modèle User et la base de données SQLite.

Les 4 opérations

Opération Route Méthode HTTP Action en base
Create /users/create GET + POST db.session.add() + commit()
Read /users GET User.query.all()
Update /users/<id>/edit GET + POST Modifier l'objet + commit()
Delete /users/<id>/delete POST db.session.delete() + commit()

CREATE — Ajouter un utilisateur

from werkzeug.security import generate_password_hash
from models import User

@app.route("/users/create", methods=["GET", "POST"])
@login_required
def create_user():
    if request.method == "POST":
        name = request.form.get("name", "").strip()
        email = request.form.get("email", "").strip()
        role = request.form.get("role", "student")
        password = request.form.get("password", "")

        if not name or not email or not password:
            flash("Tous les champs sont obligatoires.")
            return render_template("users/create.html", name=name, email=email)

        if User.query.filter_by(email=email).first():
            flash("Cet email existe déjà.")
            return render_template("users/create.html", name=name, email=email)

        user = User(
            name=name,
            email=email,
            role=role,
            password_hash=generate_password_hash(password),
        )
        db.session.add(user)
        db.session.commit()
        flash("Utilisateur créé.")
        return redirect(url_for("list_users"))

    return render_template("users/create.html")

READ — Lister

@app.route("/users")
@login_required
def list_users():
    users = User.query.all()
    return render_template("users/list.html", users=users)

UPDATE — Modifier

@app.route("/users/<int:user_id>/edit", methods=["GET", "POST"])
@login_required
def edit_user(user_id):
    user = db.session.get(User, user_id)
    if not user:
        flash("Utilisateur introuvable.")
        return redirect(url_for("list_users"))

    if request.method == "POST":
        user.name = request.form.get("name", "").strip()
        user.email = request.form.get("email", "").strip()
        user.role = request.form.get("role", "student")
        db.session.commit()
        flash("Utilisateur mis à jour.")
        return redirect(url_for("list_users"))

    return render_template("users/edit.html", user=user)

DELETE — Supprimer

@app.route("/users/<int:user_id>/delete", methods=["POST"])
@login_required
def delete_user(user_id):
    user = db.session.get(User, user_id)
    if user:
        db.session.delete(user)
        db.session.commit()
        flash("Utilisateur supprimé.")
    else:
        flash("Utilisateur introuvable.")
    return redirect(url_for("list_users"))

Types de champs HTML

<input type="password" name="password">

    <option value="student">Élève</option>

Récupération côté Flask

name   = request.form.get("name", "").strip()
email  = request.form.get("email", "").strip()
role   = request.form.get("role")
active = request.form.get("active") == "1"

Règles importantes

  1. method="POST" pour envoyer des données
  2. name="..." obligatoire sur chaque champ
  3. Validation côté serveur toujours (pas seulement required HTML)
  4. POST pour DELETE — jamais un simple lien <a href>
